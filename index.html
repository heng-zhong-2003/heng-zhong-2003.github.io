Research Experience

3. Zombie: A software caching and rematerialization system

Advisor: Marisa Kirisame, PhD Candidate

Design and implemented a software caching system, and a purely functional language built thereon. This system allows programs to continue running (albeit slower) while using much less memory than necessary to store all of the program's heap-accessible values.

Detailed results are shown in paper [1]. The project repository itself is stored at https://github.com/MarisaKirisame/zombie and https://github.com/MarisaKirisame/TVirus

2. Circuit Complexity of Universal Hashing

Advisor: Dr. Yuan Li

This project intends to find tight bounds on the circuit complexity of universal hashing. Still in progress. Phased results have been made. This is a long-term project.

1. Design and Optimization of Graph Isomorphism Algorithms

Advisor: Dr. Haibin Kan

  • Significantly optimized the VF algorithm using the method of canonical labeling of graphs, reducing the candidate vertices in the matching procedure using WL refinement.
  • Result: Non-isomorphic cases can be detected in O(n^2) time during preprocessing with only probability exp(-cn log n/log log n) of failure. Moreover, in the failure and isomorphic cases, the average running time in experiment has decreased by more than a half.
